Resumo

As tabelas acima demonstram como organizar dados de forma clara e eficiente utilizando HTML e CSS. A primeira tabela apresenta informações sobre alunos, enquanto a segunda tabela lista produtos com seus respectivos preços e quantidades.

Utilizar tabelas é uma ótima maneira de estruturar dados tabulares, facilitando a leitura e compreensão das informações apresentadas.

Experimente personalizar as tabelas com diferentes estilos e cores para melhor se adequar ao design do seu site!

<table>: Define uma tabela.

<caption>: Fornece um título ou legenda para a tabela.

<thead>: Agrupa o cabeçalho da tabela.

<tr>: Define uma linha na tabela.

<th>: Define uma célula de cabeçalho em uma tabela.

<tbody>: Agrupa o corpo da tabela.

<td>: Define uma célula de dados em uma tabela.

Ir para a página de formulários







































  • <form>: Container de formulário.
  • <input>: Campo de entrada.
  • <label>: Rótulo para um elemento de formulário.
  • <textarea>: Área de texto multilinha.
  • <button>: Botão clicável.
  • <select>: Menu suspenso.
  • <option>: Opção dentro de um menu suspenso.
  • <fieldset>: Agrupa elementos relacionados em um formulário.
  • <legend>: Título para um grupo de elementos em um formulário.
  • <datalist>: Lista de opções pré-definidas para um campo de entrada.
  • <output>: Resultado de um cálculo ou ação em um formulário.
  • <progress>: Barra de progresso.
  • <meter>: Medidor de valor escalar.
  • required: Atributo que indica que um campo é obrigatório.
  • placeholder: Atributo que fornece uma dica sobre o valor esperado em um campo de entrada.
  • autocomplete: Atributo que indica se o navegador deve preencher automaticamente um campo de entrada.
  • novalidate: Atributo que desativa a validação do formulário.
  • formaction: Atributo que especifica a URL para onde os dados do formulário serão enviados.
  • formenctype: Atributo que especifica o tipo de codificação dos dados do formulário.
  • formmethod: Atributo que especifica o método HTTP para enviar os dados do formulário.
  • formtarget: Atributo que especifica onde exibir a resposta após o envio do formulário.

Lista Ordenada

Uma lista ordenada é uma lista numerada, onde cada item é precedido por um número ou letra. É útil quando a ordem dos itens é importante.

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3
