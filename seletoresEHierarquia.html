Guia de Seletores e Hierarquia CSS

Tipos de Seletores

Seletor Universal (*): Aplica estilos a todos os elementos.

Seletor de Elemento (p, div, h1): Seleciona elementos HTML específicos.

Seletor de Classe (.classe): Seleciona elementos com uma classe específica.

Seletor de ID (#id): Seleciona um elemento único com ID específico.

Para ver a implementação dos seletores, inspecione a página.

Hierarquia e Especificidade

2. IDs

3. Classes, atributos e pseudo-classes
